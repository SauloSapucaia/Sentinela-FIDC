1. ISOLATION FOREST (IF) — O ALGORITMO ESCOLHIDO

O Isolation Forest é um algoritmo de detecção de anomalias não-supervisionado que funciona dividindo aleatoriamente o espaço de features até isolar cada ponto.

Por que ele é perfeito para fraudes?
• Anomalias (fraudes) são raras → são isoladas com muito menos divisões
• Não precisa de dados rotulados (ideal para o mundo real de FIDC)
• Extremamente rápido em alta dimensionalidade (depois do PCA)
• Robusto a outliers e escalonamento 
# ETAPA 8 — Isolation Forest + LOF (código real do Sentinela_FIDC.py)

    # 1. Features selecionadas (20+ variáveis comportamentais + temporais)
    features_iforest = ['vlr_nominal', 'desvio_ticket_pagador', 'desvio_ticket_beneficiario',
                        'freq_dupla_pag_benef', 'share_financeiro_benef', 'z_score_setorial',
                        'z_score_uf', 'dias_desde_ultimo_boleto_pagador', ...]

    X_iforest = df_sentinela[features_iforest].copy()

    # 2. Pré-processamento robusto
    X_iforest = X_iforest.replace([np.inf, -np.inf], np.nan).fillna(0)
    for col in X_iforest.columns:                                      # Winsorização P1-P99
        X_iforest[col] = X_iforest[col].clip(lower=X_iforest[col].quantile(0.01),
                                            upper=X_iforest[col].quantile(0.99))

    scaler = RobustScaler()
    X_scaled = scaler.fit_transform(X_iforest)

    pca = PCA(n_components=15, random_state=42)                        # Reduz dimensionalidade
    X_pca = pca.fit_transform(X_scaled)

    # 3. Contamination rate adaptativo (baseado na EDA)
    sinais_fraude = (df_sentinela['flag_duplicata_transacional'].mean() +
                    df_sentinela['flag_overpayment_alto'].mean() + ...) / 5
    contamination_rate = max(0.008, min(0.025, sinais_fraude * 1.8))

    # 4. Modelo principal
    modelo_if = IsolationForest(n_estimators=300, contamination=contamination_rate,
                                max_samples='auto', random_state=42, n_jobs=-1)
    modelo_if.fit(X_pca)

    # 5. Ensemble com Local Outlier Factor (densidade local)
    modelo_lof = LocalOutlierFactor(n_neighbors=20, contamination=contamination_rate, n_jobs=-1)
    lof_scores = modelo_lof.fit_predict(X_pca)

    # 6. Flag final da IA
    df_sentinela['flag_fraude_ia'] = np.where(
        (modelo_if.predict(X_pca) == -1) | (lof_scores == -1), 1, 0)
